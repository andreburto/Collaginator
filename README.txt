COLLAGINATOR 2.0
--------------------------------------------------------------------------------

ABOUT / USAGE:

A work in progress program to create desktop images for computers based on input
from the user. Currently has two usage modes:

Squares:
	java Collaginator --type square --width [int] --height [int]

Yvonne
	java Collaginator --type yvonne --width [int] --height [int] --random [boolean]
